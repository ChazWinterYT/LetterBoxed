Stitch-uations

A LetterBoxed Word Puzzle Game

About the Game

Stitch-uations is a LetterBoxed word puzzle game inspired by The New York Times LetterBoxed puzzle. Challenge yourself to solve word puzzles by connecting letters around a game board to form words.

Each word must start where the previous word ended. Try to solve the puzzle using as few words as possible!

How to Play

  1. You're given letters arranged around a square game board
  2. Form words by connecting letters in sequence
  3. Each new word must start with the last letter of the previous word
  4. Use all letters on the board to complete the puzzle
  5. Try to solve using as few words as possible!

Features

  • Play Today's NYT Puzzle - Solve the official New York Times LetterBoxed puzzle daily
  • Game Archive - Access and play any past NYT LetterBoxed puzzle
  • Custom Games - Create your own LetterBoxed puzzles with custom letters and board sizes
  • Random Games - Play randomly generated puzzles in multiple languages
  • Multi-language Support - Play in English, Spanish, French, German, Hebrew, Japanese, Russian, and more
  • Progress Tracking - Your game progress is automatically saved
  • Solution Statistics - See how your solution compares to others

Game Types

Official NYT Games: Play the daily puzzle from The New York Times LetterBoxed game.

Custom Games: Create puzzles by entering letters manually or using seed words to generate a board.

Random Games: Generate random puzzles guaranteed to have solutions, available in multiple languages.
